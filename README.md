# Face Analyzer — AI Facial Feature Analysis

An AI-powered facial feature analysis app built with FastAPI + React.

## Stack
- **Backend**: FastAPI + MediaPipe + OpenCV (Python 3.10)
- **Frontend**: React + Vite

## Run Locally

### Backend
```bash
cd backend
pip install -r requirements.txt
uvicorn main:app --host 0.0.0.0 --port 8000
```

### Frontend
```bash
cd frontend
npm install
npm run dev
```
